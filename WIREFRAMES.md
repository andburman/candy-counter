## Spacing System

```
Page Level:
- Container: max-w-6xl (1152px)
- Padding: px-4 sm:px-6 lg:px-8
- Vertical: py-8
- Section spacing: space-y-6 lg:space-y-8

Component Level:
- Card gap: gap-6 (24px) - internal card spacing
- Metrics grid gap: gap-4 (16px)
- Side-by-side gap: gap-6 (24px)
- Chart margins: 10px (left/right/top), 30px (bottom)
```
